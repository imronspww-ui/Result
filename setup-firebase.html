วิธีตั้งค่า Firebase สำหรับ SOP Image Sharing
ทำครั้งเดียว · ใช้ได้ตลอด · ฟรี
สิ่งที่ต้องมี
Google Account ฟรี 100% ไม่ต้องใส่บัตรเครดิต
ใช้เวลาประมาณ 10 นาที
1
สร้าง Firebase Project
ไปที่ console.firebase.google.com
  1. เปิด console.firebase.google.com → Login ด้วย Google
  2. คลิก "Add project" หรือ "Create a project"
  3. ตั้งชื่อ Project เช่น mrdiy-agent
  4. Google Analytics → ปิดได้ (Toggle off) → คลิก "Create project"
  5. รอสักครู่ → คลิก "Continue"
2
เปิดใช้ Realtime Database
สำหรับเก็บข้อมูลว่ารูปไหนอยู่หัวข้อไหน
  1. เมนูซ้าย → Build → Realtime Database
  2. คลิก "Create Database"
  3. เลือก Location: Singapore (asia-southeast1)
  4. Security rules → เลือก "Start in test mode" → คลิก Enable
⚠️ Test mode หมดอายุใน 30 วัน — ต้องแก้ Rules ให้ถาวรในขั้นตอนที่ 4
3
สมัคร ImgBB (เก็บไฟล์รูปจริง)
ฟรี ไม่ต้องผูกบัตรเครดิต — ใช้แทน Firebase Storage
ℹ️ Firebase Storage ปัจจุบันต้องผูกบัตรเครดิต (Blaze Plan) จึงใช้ ImgBB เก็บไฟล์รูปแทน — ฟรี 100% ไม่ต้องบัตร
  1. เปิด api.imgbb.com
  2. คลิก "Get API Key" → Login ด้วย Email หรือ Google
  3. คัดลอก API Key ที่ได้ (ตัวอักษร+เลขยาวๆ)
  4. เปิดไฟล์ imgbb-config.js ใน D:\Agent → วาง Key แทนค่าเดิม
4
ตั้ง Rules ให้ใช้งานได้ถาวร
ทำทั้ง Realtime Database และ Storage
Realtime Database Rules:
เมนูซ้าย → Realtime Database → แท็บ Rules → แก้เป็น:
// Realtime Database Rules
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
คลิก Publish
5
คัดลอก Config มาใส่ไฟล์
แก้ไขไฟล์ firebase-config.js
  1. Firebase Console → คลิก ⚙️ (Project Settings) ด้านบนซ้าย
  2. เลื่อนลงมาที่ "Your apps" → คลิก </> (Web)
  3. ตั้งชื่อ App เช่น mrdiy-web → คลิก Register app
  4. จะเห็น firebaseConfig → คัดลอกทั้งหมด
  5. เปิดไฟล์ firebase-config.js ใน D:\Agent → วางแทนค่าเดิม
ตัวอย่าง firebase-config.js ที่ถูกต้อง:
const firebaseConfig = {
  apiKey: "AIzaSyB-xxxxxxxxxxxxxxxxxxxxxxxxx",
  authDomain: "mrdiy-agent.firebaseapp.com",
  databaseURL: "https://mrdiy-agent-default-rtdb.asia-southeast1.firebasedatabase.app",
  projectId: "mrdiy-agent",
  storageBucket: "mrdiy-agent.appspot.com",
  messagingSenderId: "123456789012",
  appId: "1:123456789012:web:xxxxxxxxxxxxxxxx"
};
6
อัปโหลดไฟล์ทั้งหมดขึ้น GitHub
หลังใส่ Config เรียบร้อยแล้ว
  1. Login github.com → คลิก New repository
  2. ชื่อ: mrdiy-agent → เลือก Public → Create repository
  3. คลิก "uploading an existing file"
  4. ลากทั้งโฟลเดอร์ D:\Agent ขึ้นไป (รวม lib/, firebase-config.js, imgbb-config.js)
  5. คลิก "Commit changes"
  6. ไปที่ Settings → Pages → Source: Deploy from branch → main → / (root) → Save
  7. รอ 2-3 นาที → ได้ลิงก์ เช่น https://imron-s.github.io/mrdiy-agent
✅ เสร็จแล้ว! แชร์ลิงก์ให้ทีมได้เลย — รูปที่อัปโหลดในหน้า SOP ทุกคนจะเห็นพร้อมกัน
